# 🎙️ Transcribe AI

A simple web app that converts audio and video into text using Whisper — directly in the browser.

## 🚀 Features

* Upload audio/video files
* Select language and model
* Transcribe speech to text
* View transcript and JSON output
* Copy or download results
* Save transcription history

## 🛠️ Tech Stack

* HTML, CSS, JavaScript (Vite)
* Whisper (browser-whisper)
* WebGPU / WebAssembly

## ⚙️ Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/transcribe-ai.git
cd transcribe-ai
npm install
npm run dev
```
