About This Project

VoiceScribe AI is built as a web application project to demonstrate how artificial intelligence can be integrated into a browser interface for practical use. It allows users to upload audio or video files and generate transcript output directly in the browser.

What it does Converts spoken content from audio or video files into text.
How it works Uses a pre-trained Whisper-based model for inference, not custom model training.
Why it is useful Helpful for lectures, interviews, meetings, subtitles, notes, and accessibility.
